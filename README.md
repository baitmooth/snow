Snow
---

a simple monotone icon pack forked from [Frost](https://github.com/dkanada/frost). It is FOSS and provides mostly icons for open-source apps on F-Droid (but there are many other apps in this pack too). The design aims for simplicity with easily recognizable shapes.

Built with [Candybar-FOSS](https://github.com/Donnnno/candybar-foss), Snow includes support for a wide range of launchers, including:
**ABC, Action, ADW, Apex, Atom, Aviate, BlackBerry, Evie, Flick, Go EX, Holo, Hyperion, Lawnchair, Lucid, Niagara, Nova, Pixel, Poco, Smart, Solo, Square, ZenUI**, and maybe more.

The installation process depends on your launcher and can usually be found within the launcher's settings. If your launcher supports it you can apply the icons from the Snow app itself.
*Note: Current versions of LineageOS may (still) have an issue in Trebuchet that makes it incompatible with Snow. If you are using Trebuchet please let me know if it still persists.*

## Contributing
I appreciate your icon requests! Please keep in mind that maintaining this project is a voluntary effort, so adding new icons may take some time. I value your patience and welcome any contributions. If you have the skills and interest, you can contribute by creating icons or submitting requests. Here are some ways to get involved:


### Submitting icon requests

#### Snow's built-in feature

You can submit a limited number of icons directly through the app. The submit count resets with each new release.
1. Open Snow.
2. Open the navigation drawer and select `Icon Request`.
3. Choose the missing apps you wish to submit.
4. Press the plane symbol and choose an app to send the request via email.
5. Review the information and send the email.

**Privacy Note:** We do not share any user information. Only the necessary app details (package names and activities) required for the icon request are published. Any personal information will be stripped.

In case you need to clean the cache, there is an option named `Clear Icon Request Data` in `Settings`.

#### IconRequest (External Tool)

Another way to request icons is to use the [IconRequest](https://github.com/Kaiserdragon2/IconRequest/) app.

1. Install *IconRequest*.
2. If you have icons that broke after an update:
     1. Select the apps to report.
     2. Tap the copy icon and [open a new issue](https://github.com/baitmooth/snow/issues/new?assignees=baitmooth&labels=activity+changed&template=broken-icons.md&title=Broken+Icons).
     3. Paste the clipboard content and submit.
3. If you have apps whose icons are missing from Snow:
     1. Select the apps to request.
     2. Tap the save icon and [open a new issue](https://github.com/baitmooth/snow/issues/new?assignees=baitmooth&labels=new+icons%2C+small+request&template=requesting-new-icons.md&title=New+Icon+request).
     3. Attach the .zip file created by the app and submit.

### Making icons

Making icons is fun and **you don't even have to know how to code**. Just install a vector graphic program like [Inkscape](https://inkscape.org/en/) or [Vector Asset Creator](https://play.google.com/store/apps/details?id=com.inglesdivino.vectorassetcreator).
